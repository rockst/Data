# 資料採擷的演算法 #

* C4.5
	* 分類決策樹
* K-Means
	* 分裂法的分群演算法
	* k < n
* SVM (Support Vector Machine)
	* 監督式學習，應用於統計分析與回歸分析中
* Apriori
	* 採擷布林連結規則頻繁項集演算法
	* 逐層搜索的反覆運算方式
* EM (Expectation-Maximization)
	* 最大期望值
	* 反覆運算方式
* PageRank
	* Google: Larry Page
	* 根據網站的外部連結與內部連結的數量與品質，衡量網站的價值
* AdaBoost
	* 反覆運算方式
	* 弱分類器->強分類器
* kNN (k-Nearest Neighbor) k 最近鄰
	* 文字分類演算法
* NBC (Naive Bayesian Model) 單純貝氏演算法
	* 以機率的形式表達所有形式的不確定
* CART (Classification and Regression Trees) 分類與回歸樹
	* 二元樹
