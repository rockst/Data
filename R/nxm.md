# n x m 資料集 #

* n : 資料的行 (rows) 即觀測點(observations)的數量
* m : 資料的列 (columns) 即變數(variables)的數量
* n x m 表示資料的維度

# 資料的分類 #

* 定量資料 Quantitative Data
	* 身高、體重
* 定性資料 Qualitative Data
	* 定類別資料
	* 定序資料
	* 定距資料
	* 定比資料

## 定性資料可進行的運算 ##

| | 定類別 | 定序 | 定距 | 定比 |
| - | - | - | - | - |
| 分類(=,1) | v | v | v | v |
| 排序(<,>) | | v | v | v |
| 間距(+,-) | | | v | v |
| 比值(',) | | | | v |
